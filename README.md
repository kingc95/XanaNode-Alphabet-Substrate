# XanaNode Alphabet Substrate

This is a tiny sample federated substrate.

It has a start trail, twenty-six letter nodes, one source node, typed relationships, and bundled SVG media for every letter. Its job is to make testing boring in the best possible way: clone it, open it, build it, federate it, and verify that files, nodes, relationships, media, and archives all travel together.

The portable archive is written to `dist/` as a `.substrate` file.
